🔑 APIs & Keys Management

Step-by-step instructions for every key — no experience needed 👇

🔒 All keys are saved only on your device — never sent to any server. You are 100% in control.
📖 New to APIs? Read this first — How to get each key
An API key is just a password that lets two apps talk to each other. It's free on most platforms. Here's exactly how to get each one:
📍 Geolocation Key (IPinfo.io — FREE)
  1. Open ipinfo.io/signup in your browser
  2. Sign up with your email (it's free, no credit card)
  3. After login, go to your Dashboard
  4. Your Access Token is shown at the top — copy it
  5. Paste it in the Geolocation field below and click Save Key
✅ Free: 50,000 requests/month — more than enough
📈 Analytics Key (Google Analytics — FREE)
  1. Go to analytics.google.com
  2. Sign in with your Google account
  3. Click Start Measuring → fill in your account & property name
  4. Choose Web as your platform, enter any URL
  5. Your Measurement ID (looks like G-XXXXXXXXXX) is shown — copy it
  6. Paste it in the Analytics field below and click Save Key
✅ 100% Free forever from Google
💰 AdMob Key (Google AdMob — FREE)
  1. Go to admob.google.com and sign in
  2. Click Sign Up for AdMob if first time
  3. After setup, go to Apps → Add your app
  4. Your Publisher ID is shown at top (ca-app-pub-XXXXX)
  5. For the API key: go to console.cloud.google.com → APIs → Enable AdMob API → Credentials → Create API Key
  6. Copy the key and paste below
✅ Free — you earn money from showing ads in your apps
🏪 App Store Key (Google Play / Apple)
  1. Google Play: Go to play.google.com/console → Setup → API access → Link to Google Cloud → Create Service Account → Download JSON key
  2. Apple App Store: Go to appstoreconnect.apple.com → Users & Access → Keys → Generate API Key → download the .p8 file
